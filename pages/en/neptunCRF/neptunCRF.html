What is the program about?

  • It is a program that helps you register for courses and exams faster in Neptun.

Where can i download it?

  • At the following link

How does it work?

With your browser do the stepps below

  1. Open your prefered browser.
  2. Go to your Neptun page.
  3. Log in.
  4. Go to the course registration page.
  5. Open the course you want to register for.
  6. Tick the checkbox next to the course.
  7. Do this for all the courses you want to register for.
    1. Put the pages in one browser instance to different pages or
    2. Open the pages in different windows desktops
  8. Your set up is ready online.

With the program and do the stepps below

  1. Open the program.
  2. On your first time using the program do the stepps below.
    1. Press 1 to enter settings.
    2. Press 1 to enter your NeptunCRF email.
    3. Press 2 to enter the password you recived to your email.
    4. Press 5 to enter your prefered looping method (browsers/ desktops).
    5. Press 8 to save the settings.
  3. Press 0 to change what you want the program to do.
  4. Press 6 to set up when to start the program.
  5. Press 7 to set up how to start the program (after the delay/ if the clock hits the time, set above).
  6. Press 8 to save the settings.
  7. (Or press 9 to discard changes.)

I strognly suggest to test your setup with button 2 (start the neptun run) before you set up the program to start automatically.

If your test fails you can try to fix it by doing the stepps below.

The program fails to click the appropriate button

  1. Press 1 to enter settings.
  2. Press 3 to change the targetting method.
  3. Press 4 to enter the x position of the button to click.
    1. Open the course registration page.
    2. Open a course / click on the + sign.
    3. Press 4 to enter the x position of the button to click.
      • Make sure that the part of the button your hovering is entirely gray.
      • Make sure that no other buttons are in the way looking from the top.
      • Make sure that in course registration mode more then 5 gray pixels are visible.
    4. Press any button after your cursor is hovering above the desired button.
  4. Press 8 to save the settings.

The program fails to connect to the server

  • Try connecting to another network. (SZTE-guest does not work.)
